# Imperial AI Agent Hackathon Submission

Project: Agent Desk

Agent Desk is a paid AI-work skill marketplace on Solana devnet. A buyer agent can ask for a task
brief, receive a machine-readable price challenge, pay with a Solana Pay reference, and receive a
hash-receipted delivery packet that another AI agent can execute or verify.

## What It Sells

`agent-desk-brief`: a copy-pasteable work packet for AI agents. The paid response contains:

- objective
- deliverables
- acceptance criteria
- verification evidence requirements
- reusable skill prompt
- sha256 receipt

## Working Paths

No-Docker quickstart:

```sh
cd examples/agent-economy/quickstart
npm install
npm run server
```

In another terminal:

```sh
cd examples/agent-economy/quickstart
npm run buyer
```

Full CoralOS market path:

```sh
docker compose up -d coral
bash build-agents.sh
docker build -f coral-agents/verifier-agent/Dockerfile -t verifier-agent:0.1.0 .
npm run freelancer
```

The full path uses the existing CoralOS market loop:

```text
WANT -> BID -> AWARD -> DEPOSITED -> DELIVERED -> VERIFIED -> RELEASED
```
